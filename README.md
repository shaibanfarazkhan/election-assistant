# Election Guide Assistant

## 📌 Project Description
This is a full stack web application built using Node.js, Express, HTML, CSS, and JavaScript.  
It helps users get election-related guidance and information.

## 🚀 Features
- Interactive UI
- Backend API using Express.js
- Real-time data handling
- Simple and user-friendly design

## 🛠️ Tech Stack
- Node.js
- Express.js
- HTML
- CSS
- JavaScript

## ▶️ How to Run
1. Clone the repository
2. Run `npm install`
3. Run `node server.js`
4. Open `http://localhost:3000`
